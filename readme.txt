This Label List Widget helps you to create a nice category or archive list with labels.

== Description ==

With this plugin you can easily set up sidebar list for your categories and archives.

= Features =
* Widget supports two types of lists - Archive list and Categories list
* In Archive list you can choose from three periods - daily, monthly or yearly
* In Categories list default order is alphabetical, you can change it to order by post count (ascend or descend)
* You can set the number of lines to show in the list
* You can style the list with custom font sizes and font and background colors
* You can add your own CSS styles


== Installation ==

1. Upload category-archive-label-list-widget to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
